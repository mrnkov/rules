Точное совпадение домена (без поддоменов)
- DOMAIN,rutracker.org
Совпадение домена и ВСЕХ его поддоменов (самый нужный вариант)
- DOMAIN-SUFFIX,twitter.com
- DOMAIN-SUFFIX,x.com
Если в имени домена есть определенное слово (осторожно, может зацепить лишнее, например my-google-search.com)
- DOMAIN-KEYWORD,google
Регулярное выражение (сложная маска домена) (например, всё, что содержит слово netflix)
- DOMAIN-REGEX,^.*netflix.*$
Конкретный IP-адрес IPv4 (обязательно писать /32 на конце)
- IP-CIDR,1.1.1.1/32
- IP-CIDR,8.8.8.8/32
Целая подсеть (диапазон) IPv4-адресов
- IP-CIDR,192.168.0.0/16
- IP-CIDR,10.0.0.0/8
IPv6 адреса (конкретный адрес или подсеть)
- IP-CIDR6,2606:4700::1/128
Привязка к номеру ASN (автономной системы провайдера) (например, так можно завернуть сразу все сервера Telegram)
- IP-ASN,59930
Все запросы, идущие на определенный порт (например, нестандартный SSH)
- DST-PORT,2222
Все запросы с определенного внутреннего порта устройства
- SRC-PORT,7777
